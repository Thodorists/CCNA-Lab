# AdvancedLab1

## Περιγραφή
Σε αυτό το lab, δημιουργήθηκε ένα δίκτυο με έναν router και τρία switches, συνδεδεμένα με PCs σε διαφορετικά subnets. Στόχος ήταν η εξάσκηση σε routing, VLANs και βασικές ρυθμίσεις δικτύου.

## Ρυθμίσεις IP

| Συσκευή | Διεύθυνση IP    | Μάσκα Υποδικτύου | Default Gateway |
|---------|-----------------|------------------|-----------------|
| Router  | 10.255.255.254  | 255.0.0.0        | -               |
| PC1     | 10.255.255.1    | 255.0.0.0        | 10.255.255.254  |
| PC2     | 172.16.255.1    | 255.255.0.0      | 172.16.255.254  |
| PC3     | 192.168.0.1    | 255.255.255.0    | 192.168.0.254  |
